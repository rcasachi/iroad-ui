# Spacing

Spacing patterns are essential in good interface design. They can build pace, consistency, readability, and build trust in the brand.

Spacing in digital design helps create correct hierarchy and focus between screen elements. To facilitate use and standardize spacing values, a linear scale is used as a basis.
